# RRT/RRT*

Python implementation of the RRT/RRT* algorithms using pygame. 

## Prerequisites
- [pygame](https://www.pygame.org/wiki/GettingStarted)

## Demo
Just run `python3 RRT.py` from terminal.

- Max Iterations: 500
- 2% Goal Bias
- Algorithm stops once goal state is reached.
